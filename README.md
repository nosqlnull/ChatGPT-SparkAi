## SparkAi系统 新一代 AIGC 系统，一站式 AI B/C 端解决方案 使用 Nestjs 和 Vue3 搭建的 AIGC 商业化运营ChatGPT网站系统和AIGC系统

### 一站式 AI B/C 端解决方案，AI大模型提问、AI绘画、AI视频生成（最新发布）、文档分析、多模态识图理解、TTS & 语音识别对话、AI换脸、支持AI智能体应用（支持GPTs应用+Prompt自定义预设应用）、插件系统、AI音乐生成、一站式系统！

## 当前特色支持功能模块
- 支持GPT-3.5/GPT-4所有模型版本
- 支持多模态模型 & 联网
- 支持国内 & 三方AI全模型
- 自定义AI智能体（GPTs应用/Prompt预设应用）
- PDF文档分析、识图理解、插件系统
- TTS & 语音识别对话
- 专业AI视频生成（最近更新发布）
- Suno文生音乐模型（Chat形式）
- 支持DALL-E2 、 DALL-E3绘画
- 支持Midjourney绘画
- 支持以图生图（垫图）、局部重绘
- 支持文生图、混图、AI换脸
- 智能思维导图生成
- 全套卡密系统支持
- 完善的后台管理系统支持
- 即将支持 GPT5.0、Sora（待官方发布）

## 系统技术架构
- 前端： Vite + vue3 + ts + NaiveUI + Tailwindcss 
- 管理端：Vite4 + Vue3 + Element-Plus
- 服务端（后端）：Nodejs + NestJs
- 数据支持： Mysql5.7(+) + Redis
- 运行环境：Linux、Windows、MacOs（推荐使用Linux）
- 数据存储：本地存储 | 阿里云oss | 腾讯云cos | chevereto图床

### 功能模块介绍：
- [x] 使用 Nodejs + NestJs 服务端渲染技术, 支持高并发（万级甚至千万级用户同时请求使用）；提供更高的可用性和稳定性。
- [x] 支持宝塔常规部署和Decker一键部署方式，所有对接配置均可在后台界面上完成。
- [x] 支持OpenAI官方API + One API 等中转对接使用（即支持OpenAI全模型、三方大模型、国内全模型）；支持 OpenAI / Azure OpenAI / Anthropic Claude / Google Gemini / Midjourney / 讯飞星火 / 百川 AI / Moonshot / 智谱 ChatGLM / 通义千问 / 云雀大模型 / 腾讯混元 / 360 智脑等模型, 适配 LocalAI / Ollama 等模型 (LLaMa / RWKV / SDXL )，支持openai发布最新gpt-4o大模型对接使用.....
- [x] 自定义模型对接系统：所有模型走OpenAI的输入格式，当OpenAI发布新的语言模型，可后台第一时间添加，无须系统每次开发对接才能使用！同理支持对接了其他三方模型的中转系统，即支持OpenAI全模型、三方大模型、国内大模型 最新发布模型快速可用。对话模型自定义分类、自定义名称自定义排序、以及部分功能自定义。
- [x] 登录系统：支持 邮箱 | 手机短信 | 微信扫码 等不同登录注册方式供选择。
- [x] AI视频生成独立功能，支持文生视频/图生视频（最新新增）
- [x] 支持任何 OpenAI 格式中转API, 自研渠道均衡负载和分配算法, 支持多渠道高并发调用管理， 支持多 API Key 轮询！ (优先级/权重/状态管理)。
- [x] 对话会话隔离、云端存储、支持多设备共享数据！
- [x] 多模态模型支持： 识图、文档分析支持。支持自定义对话模型开启多模态支持。
- [x] 支持联网模式、可对模型进行扩展搜索当前网络实时内容总结。
- [x] 支持AI智能体应用：GPTs应用+Prompt自定义预设应用；GPTS：支持GPTS后台自定义添加、也可以全站搜索 = 官方搜索。
- [x] 语音识别 & TTS  , 支持 OpenAI / Azure 等语音识别 & TTS 模型，支持 Whisper & TTS 格式中转；支持TTS对话输入回复模式。
- [x] 支持 Midjourney全功能, 支持 Imagine / Upscale / Vary / Zoom Out 等众多功能， 支持 Seed 获取。支持 中转API / 支持局部重绘 (Vary Region)等等。
- [x] 专业绘画支持文生图、图生图、垫图混图、角色一致参考图、风格一致参考图生成等、支持AI换脸、支持混合图片。
- [x] Dall-E-2/3绘画支持、支持所有参数并享有单独模块。
- [x] 大 Markdown 语法支持 (支持 代码高亮 / LaTeX 公式 / Mermaid 思维导图 / 图表绘制), 支持集成对话绘图模型 (DALL-E / Stable Diffusion / Midjourney 等)。
- [x] 开箱即用的文档解析服务, 支持 Pdf / Docx / Xlsx / 音频 / 图片等文件类型解析。
- [x] 存储系统：支持本机、阿里云OSS、腾讯云cos、Chevereto图床多种类型。
- [x] 会员系统： 提供普通模型积分、高级普通模型积分、绘画积分三种货币类型、支持各类商品自定义扣费模式与额度。支持自定义模型扣除费用类型、多种计费方式：按时间限制、无时间限制、自定义组合套餐设置 (免费 / 积分（次数）扣费 计费)。
- [x] 支持官方微信支持、易支付、码支付、虎皮椒支付等支付方式, 支持同步检查订单状态, 支持订单搜索和管理。
- [x] 商城系统：自定义商品套餐、可自定义生成永久套餐、限时套餐。
- [x] 签到系统： 支持配置每日签到赠送不同额度的奖励。
- [x] 访客系统： 支持不登录使用部分内容、可游客模式体验站内功能。
- [x] 分销系统：支持 A + B 分销模式、后台可自定义分销提成额度、可对单独用户单独设置。支持设置提现门槛选项，支持用户多种提现方式选择（支付宝、微信、银行卡等）。
- [x] 卡密系统：支持批量生产卡密供用户兑换、用户端可兑换卡密或三方卡网购买支持完备卡密兑换码体系, 支持自定义点数, 支持批量生成和卡密兑换码管理, 支持卡密兑换码查看领取用户 / 创建时间 / 领取时间等信息；删除卡密兑换码等功能。
- [x] 插件系统开放、内部支持各类插件并会持续开发更多插件。
- [x] 多端支持，自适应布局、适配PC、移动端、平板等设备。支持 Web / PWA / App / 小程序（后续开发）, UI 移动端适配, 支持明暗主题切换,  支持 Windows / MacOS / Linux / Android / iOS App。
- [x] 专业站点管理系统, 支持自定义配置网站 LOGO / 站点名称 / 页脚 / 百度统计/ AI名称 / 版权信息/ 联系方式 / 站点公告 / 提示欢迎语等多种内容等多种内容;支持设置用户初始点数, 单独用户群配置 等等。
- [x]  数据统计, 丰富美观的仪表盘统计，支持每日新增用户数量统计 / 总计和每日对话数量 / 总计和每日新加绘画数量 / 总计和每日日订单数量统计 /  等多种信息统计图表趋势展示。
- [x] 内容敏感词多种限制、支持自定义敏感词/百度内容审核等安全检测、可共同开启。支持敏感词检查记录查询统计！
- [x] 动态菜单系统：支持内嵌网页、外部链接跳转、内部路径跳转、可以自定义开关菜单、添加菜单、自定义ICON、自定义菜单类型、可使用三方站点内嵌、支持移动端PC端各自单独设置。
- [x] 管理端支持超管和普通演示账户用于分销系统、普通账号仅可读权限
- [x] 管理端支持配置关键词回复、可自定义关键词及回复内容
- [x] 推广邀请奖励，支持邀请和被邀请用户设置自定义积分奖励。
- [x] 板块添加, 支持添加自定义板块嵌入网站页面, 支持设置板块图标，使站点更加丰富多样。
- [x] 其他功能...
- [x] 更多AI能力持续开发更新中...
